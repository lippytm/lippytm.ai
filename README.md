# lippytm.ai
A AI Web3AI connection Hub with Gist, Git, GitLab, GitHub and all of my repositories.

## AI BrainKit

The **AI BrainKit** provides a standardized set of AI configuration files, connector templates, and automation patterns shared across all lippytm repositories. It lives in the [`brainkit/`](brainkit/) directory and serves as the central intelligence layer for the entire ecosystem.

- [AI BrainKit Overview](brainkit/README.md) - What it is and how to apply it to any repository
- [BrainKit Metadata](brainkit/brainkit.json) - Registry of all connected repositories and features
- [Copilot Instructions Template](brainkit/copilot-instructions.template.md) - Drop-in GitHub Copilot config for any repo
- [Connector Template](brainkit/connector.template.js) - Base connector class for platform integrations
